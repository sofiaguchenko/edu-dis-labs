# Висновки

В результаті виконання проєкту було розроблено скраппер html.
Під час розробки даного продукту було в повному обсязі реалізовано технічне завдання, за допомогою готової програми можна виконати поставлені на початку роботи задачі.
Виконання курсової роботи допомогло закріпити матеріал і навички, отримані на лекціях з курсу «Розподіленні інформаційні системи» та освоїти новий матеріал, що ще не був вивчений у ході курсу, але став у пригоді  під час вирішення конкретних завдань.
